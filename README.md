# Kurulum

Dosyayı [ESX] Klasörü içine atın
Dosya içindeki veritabanı dosyasını import edin
Aşağıdaki şekilde server.cfg nize ekleyin.
```
start instance
start mythic_progbar
start esx_propertymotels
```
